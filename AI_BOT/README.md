### Environment Setup

This project uses environment variables for API configuration.

Create a `.env` file in the project root:

VITE_GROQ_API_KEY=your_api_key  
VITE_GROQ_URL=https://api.groq.com/openai/v1/chat/completions

Note: This is a frontend-only demo. API keys are stored using environment variables and restricted via provider limits.
